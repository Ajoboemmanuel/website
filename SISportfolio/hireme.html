Why is the site / redesign / project needed? Who is the audience?

Web Development. Do you want the site to be static or dynamic?

Static HTML sites need some knowlege with HTML language and text editors to update the site, texts, pictures etc. Dynamic CMS sites, ie. WordPress, Joomla, Drupal and so on, are much easier to use and update – and won't require any HTML knowlegde. But it requires more work to develop a site for a CMS platform, so it costs more.

My personal favorite is WordPress, highly customizable and ease of use for the end-users with other cool stuff.
